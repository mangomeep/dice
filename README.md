# dice

a simple dice roller written in c

please treat it nicely otherwise it'll break :(

to download it, just type
```
git clone https://github.com/mangomeep/dice.git
cd dice
```
then to build it and run it
```
$ make
$ ./dice 1d20 2d6+6
```
or install it and run it
```
# make install
$ dice 1d100 2d10-5
```
